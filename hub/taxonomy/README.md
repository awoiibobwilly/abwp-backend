Knowledge Hub Taxonomy

The taxonomy directory contains the governance documentation for the
Knowledge Hub classification system.

Unlike fixtures, which populate the database, these documents define
the long-term rules used to classify knowledge across the entire
ABW Portfolio.

The taxonomy is considered a portfolio-wide standard and is shared by:

• Knowledge Hub
• Research
• Projects
• Insights
• Future AI Search
• Recommendation Engine
• Knowledge Graph

The taxonomy should evolve carefully to preserve consistency and
backward compatibility.
